# Hangman Game

[![Build Status](https://travis-ci.org/valterjrdev/hangman-game.svg?branch=master)](https://travis-ci.org/valterjrdev/hangman-game)

## Installation

```bash
composer install
```

## Usage

```bash
php bin/console app:game
```

Or

```bash
docker-compose run --rm app
```

## Tests

```bash
composer run test
```

## Static analysis tools

```bash
composer run static
```
